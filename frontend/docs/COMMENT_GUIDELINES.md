# 注释编写规范

## 文件头部注释

每个文件的头部应包含以下信息：
- 文件描述
- 作者
- 创建日期
- 最后编辑者
- 最后编辑时间

示例：
```vue
<!-- 
    * @FileDescription: 该文件的描述信息 
    * @Author: 作者信息 
    * @Date: 文件创建时间 
    * @LastEditors: 最后更新作者 
    * @LastEditTime: 最后更新时间
 -->
```

## 函数和方法注释
每个函数或方法应包含以下信息：

- 函数描述
- 参数描述
- 返回值描述

示例：
```vue
/** 
  * @description 方法描述 
  * @param {参数类型} 参数名称 
  * @param {参数类型} 参数名称 
  * @return 没有返回信息写 void / 有返回信息 {返回类型} 描述信息 
  */
```
